# Corpus Sources

Full inventory of sources in the Islamic Historical Corpus.
Authentication status and chain strength per source.

See the API /sources endpoint for live counts.

## Authentication Tiers

### classical_islamic
Accepted within the Islamic scholarly tradition.
Cited by Islamic scholars across centuries.

### primary_historical  
Contemporary or near-contemporary accounts.
Historically valuable, not canonically authenticated.

### scholarly_western
Modern academic scholarship.
Used for context and framing only, never as
primary voice on Islamic historical narrative.
