## Overview

This document is the complete operating manual for sub-agents handling all marketing metadata, SEO, and content distribution for Islam Stories. Every field, every platform, every format is specified. Sub-agents execute this system without human input per episode.

**Inputs sub-agents receive per episode:**
- Episode title (working title from script)
- Series name
- Primary figure(s) featured
- Battle/event name
- Era (date range)
- Key primary sources used (Al-Tabari, Al-Waqidi, etc.)
- 3-sentence episode summary
- Best Kling frame (thumbnail candidate)
- 60-second highlight clip (Shorts candidate)
- Full episode duration

**Outputs sub-agents produce:**
- YouTube: title, description, tags, chapters, thumbnail brief, end screen brief, cards brief
- YouTube Shorts: title, description, tags, caption
- TikTok: caption, hashtags, on-screen text brief
- Instagram Reels: caption, hashtags, on-screen text brief
- Facebook: post copy, hashtags

## Part 2: YouTube — Long Form Episode

### 2.1 Title Formula

**Rule:** Primary keyword in first 40 characters. Intrigue/hook element. Max 60 characters total (prevents truncation in search).

**Title Template Options (sub-agent selects best fit):**

Option A — The Question Hook:
```
[Battle/Event]: [Intriguing Question About Figure]
Example: "Battle of Yarmouk: Why Did God's Sword Go Silent?"
```

Option B — The Superlative:
```
[Figure Name]: [Superlative Claim from Primary Sources]
Example: "Khalid ibn Walid: The General Who Never Lost a Battle"
```

Option C — The Revelation:
```
The [Untold/Real/True] Story of [Figure/Battle]
Example: "The Real Story of Saladin No One Tells"
```

Option D — The Number Hook:
```
[Figure] vs [Opponent]: [Scale/Number that Shocks]
Example: "7,000 Muslims vs All of Spain: The Tariq ibn Ziyad Story"
```

Option E — The Consequence:
```
How [Figure/Battle] Changed [Region/World/History]
Example: "How Baybars Saved Islam From Total Annihilation"
```

**Title Rules for Sub-agents:**
- NEVER use clickbait that the video doesn't deliver on — YouTube penalizes short watch time
- Primary keyword must appear naturally, not forced
- Include the figure's name OR the battle name, not both (too long)
- A/B test: generate 3 title options, flag for human selection on first 10 episodes, then use best-performing pattern
- Avoid ALL CAPS except for single emphasis word maximum

## Part 7: Cross-Platform Rules

### 7.1 Platform-Specific Adaptations

**Never cross-post raw:** Each platform gets adapted content.

| Element | YouTube | TikTok | Instagram | Facebook |
|---|---|---|---|---|
| Video length | 15-20 min | 30-60 sec | 30-60 sec | 3-5 min (upload separately) |
| Caption length | 300+ words | 100-150 chars | 150-200 chars | 200-400 words |
| Hashtags | 5-8 at bottom | 3-5 end-stacked | 5-8 end-stacked | 3-5 |
| Tone | Scholarly-cinematic | Hook-first, punchy | Premium, cultural | Community-oriented |
| CTA | Subscribe + playlist | Link in bio | Link in bio | Full video link embedded |

### 7.2 Watermark Rules

Do not cross-post with platform watermarks. Instagram's algorithm deprioritizes content with TikTok watermarks, and vice versa. Download clean version without watermarks and upload natively.

- **Always remove TikTok watermark before uploading to Instagram/YouTube Shorts**
- Export clean MP4 from pipeline, upload separately to each platform
- Each platform upload = native content signal to algorithm

### 7.3 Posting Sequence (per episode launch)

```
Day 0 (Upload day):
  09:00 — YouTube long-form uploads (catches morning viewers)
  12:00 — TikTok Shorts clip 1
  13:00 — Instagram Reel (same clip, adapted caption)
  15:00 — Facebook full post with YouTube link
  18:00 — Instagram Story 1 (Reel reshare + poll)
  
Day 1:
  09:00 — TikTok Shorts clip 2 (different scene)
  19:00 — Instagram Story 2 (behind the source)
  
Day 3:
  YouTube Community post: "What did you think of [Figure]?" 
  + poll + link to episode
  
Day 7:
  TikTok Shorts clip 3 (third extracted scene if available)
```

**Why this sequence matters:** YouTube evaluates performance in first 48 hours. Cross-platform promotion in first 24 hours creates the view spike that signals the algorithm to push the video further.

## Part 9: Hook Writing System

### 9.1 The 8-Second Rule

The first 8 seconds determine whether a viewer continues watching or scrolls away. Present a strong visual hook, pose a clear problem or promise.

Every episode opens with a narration hook that sub-agents verify against this rubric:

**Strong hook criteria:**
- Poses a question the viewer doesn't know the answer to
- States a fact that contradicts what viewer assumes
- Places viewer in the scene sensory-first
- Establishes stakes within 8 seconds

**Hook Templates (for script generation sub-agent):**

The Contradiction Hook:
```
"History remembers [Figure] as [common perception]. 
[Primary source] tells a completely different story."
```

The Sensory Immersion Hook:
```
"[Year CE]. [Location]. [Sensory detail — what it 
looked/smelled/sounded like]. [Figure] is about to 
make a decision that will [consequence]."
```

The Unanswered Question Hook:
```
"In [year], [Figure] did something that historians 
have never fully explained. By the end of this 
episode, you'll understand why."
```

The Scale Hook:
```
"[Number] soldiers. [Number] days. [Geographic scale]. 
What happened at [battle] shouldn't have been possible."
```

### 9.2 Platform-Specific Hook Adaptations

The opening hook of the long-form episode becomes the basis for all platform adaptations:


## Part 11: Islam Stories-Specific SEO Notes

### 11.1 The Dual Audience SEO Strategy

Islam Stories serves two audiences with different search behaviors:

**Muslim audience searches:**
- Uses names in Arabic transliteration: "Khalid bin Walid" vs "Khalid ibn Walid"
- Searches "islamic history" + "seerah" + "companions"
- Searches figures by honorific: "Khalid RA" (Radi Allahu anhu)
- Include both transliterations in description (not title — looks unnatural)

**History enthusiast searches:**
- Uses anglicized names: "Saladin" not "Salah al-Din"
- Searches "medieval history" + "crusades" + "Islamic empire"  
- Doesn't know Islamic terminology

**Resolution:** Title uses whichever name is more searchable (usually anglicized for Western audiences). Description includes both: "Salah al-Din al-Ayyubi — known in the West as Saladin."

### 11.2 Theological Sensitivity in SEO

**Never include in titles, descriptions, or tags:**
- Names of the Prophet (PBUH) as primary hook — this is not what the channel covers, and misrepresents content
- Names of the Four Caliphs as primary subjects (Tier S figures)
- "Prophet Muhammad" as a search keyword bait when episode doesn't depict him

**This protects:**
- Theological credibility with Muslim audience
- Accurate algorithm categorization (history, not religious content)
- Channel from content that could be flagged or misrepresented

### 11.3 Language and Geographic Targeting

Islam Stories' audience spans: UK, US, Canada, Australia, Malaysia, Indonesia, Pakistan, Bangladesh, Nigeria, Egypt, Saudi Arabia, UAE.

**For maximum CPM:** Optimize for English-language search (UK/US/Canada/Australia highest CPM)

**For maximum reach:** Include episode transcripts (auto-generated) — YouTube uses these for search indexing in multiple languages

**Title language:** Always English. YouTube's algorithm distributes English titles globally.

**Description:** English only. Do not include Arabic text in description — confuses algorithm categorization.
